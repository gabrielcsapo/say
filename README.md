# saywhat  
> http://gabrielcsapo.github.io/saywhat/

# Usage

> encode

```
var Say = require('saywhat');
Say.encode('hello'); // 👬𐎛👡└💞
```

> decode

```
var Say = require('saywhat');
Say.decode('👬𐎛👡└💞'); // hello
```
